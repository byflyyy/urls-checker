# url-checker
A Python scrip to check if a url is accessible.

一个检查网页是否可以访问的python脚本,若可以访问则循环检查,可以自定义检测间隔...

初心是想测试一个网盘(115.com)的m3u8地址时效性,测试发现可以存活14400s
